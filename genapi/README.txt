In order to run this project (in its current state), you will need to run the 
following commands in terminal:

npm install express mysql2 dotenv
npm install --save-dev typescript ts-node @types/node @types/express


NOTES:
-This test API does not currently use any Phase 1 functions.
If you modify this project to include such functions, you will need
to install the necessary dependencies for those functions accordingly.
